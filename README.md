# 06-CCNA-VLAN-TRUNKING-NETWORK

## Skills Demonstrated
Designed and implemented VLAN-based segmentation across multiple switches in a distributed Layer 2 network.


Configured trunk links between switches to enable VLAN traffic propagation using IEEE 802.1Q encapsulation.

Deployed consistent VLAN configurations across multiple switching devices to maintain network uniformity.

Assigned access ports to VLANs and ensured proper end-device association within each broadcast domain.

Verified trunk operation and VLAN propagation using commands such as show interfaces trunk and show vlan brief.

Conducted connectivity testing to validate inter-switch VLAN communication and confirm isolation between different VLANs.

Demonstrated understanding of trunking concepts, VLAN tagging, and inter-switch communication in enterprise environments.

Strengthened Cisco IOS CLI proficiency through multi-device configuration and troubleshooting.
